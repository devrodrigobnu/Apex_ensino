Usando onclick do JS


O evento onclick chama a nossa função em JS dentro das tags script.
Isso ocorre quando o usuário clicar no botão!

Quando você usa innerHTML em um elemento HTML
Você pode obter o conteúdo HTML dentro desse elemento na página
Aqui está um exemplo básico de como usar innerHTML com JS.
Código fonte:
function my_function() {
document. getElementById ("elemento_id") .innerHTML = "Hello"
}
